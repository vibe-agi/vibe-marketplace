# vibe-marketplace

A community marketplace for Claude Code plugins and skills by [vibe-agi](https://github.com/vibe-agi).

## Installation

```bash
/plugin marketplace add vibe-agi/vibe-marketplace
```

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [autoresearch-creator](https://github.com/vibe-agi/autoresearch-creator) | Generate project-specific autonomous optimization skills inspired by [karpathy/autoresearch](https://github.com/karpathy/autoresearch) | 0.1.0 |

## Usage

After adding the marketplace, install any plugin:

```bash
/plugin install autoresearch-creator@vibe-marketplace
```

## Contributing

To add a new plugin to this marketplace, submit a PR modifying `.claude-plugin/marketplace.json`.
